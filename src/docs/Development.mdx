## Prerequisites

- **Node.js** v18 or higher
- **npm** v8 or higher
- **Red Hat VPN** (for API access)

---

## Initial Setup

```bash
git clone https://github.com/RedHatInsights/insights-rbac-ui.git
cd insights-rbac-ui
npm install
npm start
```

---

## Available Scripts

```bash
# Development
npm start                       # Start dev server

# Building
npm run build                   # Production build
npm run build-storybook         # Build Storybook

# Testing
npm test                        # Unit tests
npm run test-storybook:ci       # Storybook interaction tests (required before PR)
npm run test-storybook:fast     # Faster Storybook test run (no browser warmup)

# Code Quality
npm run lint:js                 # ESLint (must pass, zero errors)

# Storybook
npm run storybook               # Start Storybook dev server
```

## Commit Convention

[Conventional Commits](https://www.conventionalcommits.org/):

```
feat(workspaces): add role assignment modal
fix(groups): correct pagination reset on filter change
refactor(table-view): extract useFiltersState hook
test(user-journeys): add workspace admin journey
chore(deps): update patternfly to v6
```

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Module not found | Check import paths and webpack aliases |
| TypeScript errors | Run `npm run build` for full diagnostics |
| API errors locally | Verify Red Hat VPN connection |
| Storybook stories missing | Check file naming (`ComponentName.stories.tsx`) |
