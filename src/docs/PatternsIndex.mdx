## Contributing

When establishing a new pattern:
1. Create a focused MDX file in `src/docs/` (200–400 lines, one topic)
2. Link it from this index
3. Add it to [AGENTS.md](../../AGENTS.md) docs index
4. Include a real-world example from the codebase
5. Add a common mistakes section
